Descripción:

El EOBLBM1601HT Es un paquete de aditivos de alto rendimiento y extremadamente rentable, especialmente diseñado para la formulación de aceites de motor HDEO monogrado y multigrado. Este producto contiene componentes de alta calidad que brindan una excelente protección contra la formación de depósitos, la oxidación del aceite, el rayado de los pistones, el desgaste, el hollín y la corrosión.

Ha sido desarrollado para cumplir con el nivel de rendimiento API CF-4 e inferior. Este paquete de aditivos puede utilizarse con una amplia gama de aceites base, modificadores de viscosidad y depresores del punto de fluidez para formular aceites de motor de primera calidad con diversos grados de viscosidad.
